Projects

Python Memory Game Using tkinter

I created a memory matching game in Python, using tkinter for the GUI. The game has 12 cards and the player can flip over 2 cards at a time, revealing their respective letters. When the player selects 2 cards with matching letters, the player gains points - the player wins by matching all 6 pairs of cards.

The goal of this project was to learn Python and further my understanding of the concepts of object oriented programming.
A large part of the game are the cards, which are programmed as buttons. All 12 cards are instances of the Card class,
allowing them to share functionality while also possessing their own attributes. I designed the Card class to display
encapsulation by using accessor and mutator methods for when other objects want to modify a card's attributes.

In upgrading my project from console output to an interactive GUI I imported the random and tkinter modules as well as
a few tkinter classes. Working with the tkinter and the heirarchy of the elements, greatly helped my understanding of inheritance.
